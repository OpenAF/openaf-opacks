# Testing Asciinema Player Embedding

To add this you just need to add the following HTML:

```html
<div id="player1"></div>
<script src="/js/asciinema-emb-player.js"
        data-id="player1"
        data-record="/recs/test.cast"
        data-width="80%"
        data-autoplay="false"
        data-cols="150"
        data-rows="35"
        data-speed="1"
        data-poster="npt:0:14"
></script>
```
